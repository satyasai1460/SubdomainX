# SubdomainX
Domains Collector 

# requirments 
sudo apt install subfinder
sudo apt install amaas

# Install

```
git clone https://github.com/zapstiko/SubdomainX.git
cd SubdomainX
chmod +x subdomainx
mv subdomainx /user/local/bin
```

# Use 
subdomain domain.com
