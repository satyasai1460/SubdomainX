# SubdomainX
Domains Collector 

# requirments 
1. sudo apt install subfinder
2. sudo apt install amaas

# Install

```
git clone https://github.com/zapstiko/SubdomainX.git
cd SubdomainX
chmod +x subdomainx
sudo cp subdomainx /usr/local/bin/
```

# Use 
subdomainx domain.com
